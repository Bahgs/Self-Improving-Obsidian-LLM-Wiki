# Personal Thinking Model

This is a template for durable user or project preferences that should improve future wiki outputs.

Keep this file focused on working style, research taste, project themes, and synthesis preferences. Do not copy sensitive details, contact information, or raw memory lists.

## Current Working Model

- The user values synthesis that connects sources into reusable concepts, project ideas, decisions, and next steps.
- The user wants the vault to surface non-obvious questions and ideas, not only summarize what was asked.
- The user prefers outputs that distinguish evidence, interpretation, and speculation.
- The user wants the LLM Wiki pattern to remain the foundation, but not the ceiling.

## Output Preferences

- Prefer concrete frameworks over generic reflection.
- Preserve source-grounded claims and label weak or unsupported claims.
- Create connections across sources when they are real and useful.
- Surface strategic implications and next questions when the source warrants it.
- Keep schema changes controlled, reviewable, and grounded in repeated usefulness.
