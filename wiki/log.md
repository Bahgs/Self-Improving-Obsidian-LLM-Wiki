# LLM Wiki Log

Append every durable operation here.

Use this format:

```markdown
## [YYYY-MM-DD] operation | details
```

Examples:

```markdown
## [2026-05-12] ingest | raw/web/example.md -> [[sources/example]], [[concepts/example-concept]]
## [2026-05-12] query | "Compare X and Y" -> [[analysis/x-vs-y]]
## [2026-05-12] lint | wiki health check -> [[analysis/wiki-lint-2026-05-12]]
```
