# Generated Wiki Layer

This folder contains generated and maintained knowledge.

Use:

- `wiki/sources/` for one note per major raw source
- `wiki/concepts/` for reusable ideas, mechanisms, methods, and frameworks
- `wiki/entities/` for people, organizations, products, projects, and institutions
- `wiki/summaries/` for cross-source topic overviews
- `wiki/analysis/` for comparisons, audits, decisions, contradiction reports, and reusable query answers

Do not place raw source archives here.
