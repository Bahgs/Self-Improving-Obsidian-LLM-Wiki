# Raw Source Archive

Put source files here.

Rules:

- Agents may read files in `raw/`.
- Agents must not modify files in `raw/` during normal ingest.
- Link raw source paths as literal paths, not wikilinks.
- Do not commit private or sensitive files to a public repo.

Suggested folders:

- `raw/Clippings/` for browser-captured markdown
- `raw/web/` for cleaned web pages
- `raw/papers/` for PDFs and research papers
- `raw/books/` for book notes or excerpts you are allowed to store
- `raw/ideas/sketches/` for rough project ideas
- `raw/assets/` for images, screenshots, or supporting files
