# Obsidian CLI Skill

Use this skill when the user asks to interact with an open Obsidian vault or verify plugin/theme behavior.

Possible tasks:

- inspect active file
- run vault searches
- verify links or rendered markdown
- reload plugins
- inspect errors
- take screenshots

Do not use this skill for ordinary file edits when direct markdown editing is enough.
