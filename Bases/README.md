# Bases

Optional Obsidian Bases dashboards live here.

Suggested dashboards:

- `Brain Wiki.base`: browse all generated wiki pages.
- `Schema Proposals.base`: review proposed schema changes.
- `Source Coverage.base`: inspect source notes and raw source coverage.

This public template leaves actual `.base` files out because Base syntax and plugin support may change. Add them when your Obsidian setup is ready.
